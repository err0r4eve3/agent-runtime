# Cursor Extensions

This directory tracks the installed Cursor extension inventory for the current machine without copying the full extension payloads.

Included files:

- `extensions.json`: raw installation manifest copied from `~/.cursor/extensions/extensions.json`
- `catalog.json`: normalized extension inventory with marketplace/source links
- `packages/*/package.json`: per-extension metadata copied from each installed bundle
